AI-Powered Credit Risk Prediction System

CreditGuard-AI is an end-to-end machine learning project that predicts loan default risk using the Home Credit Default Risk dataset.

Features
Advanced feature engineering across multiple relational datasets
XGBoost-based credit risk prediction
Optuna hyperparameter optimization
Threshold optimization
Modular ML pipeline
Production-ready preprocessing
FastAPI backend (work in progress)
